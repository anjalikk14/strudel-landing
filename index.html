Every file is automatically shareable and realtime editable, accessible from anywhere.

That's the long term goal of the Strudel project. We want to build a platform where users can create files for any possible use-case, and never have to think twice about whether they can collaborate with others. And, at the same time, we want to create a realtime collaboration API for developers to easily plug into any project they are working on. Files are a fundamental building block of computing, but for some reason the file concept did not make the jump to the internet era. At Strudel, we are reinventing files by building in realtime collaboration, version control, and cloud storage for every file from the get-go. We hope to eventually become the first platform to succeed at building a truly online OS.

Strudel isn't quite ready for the limelight yet, but sign up for our mailing list to keep up to date with what we're working on!
